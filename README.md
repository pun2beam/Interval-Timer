# Interval-Timer

繰り返しインターバルタイマーは、指定した周期を停止するまで繰り返す静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 時・分・秒を選択します（初期値は2分）。
3. 「スタート」を押すと、同じ周期のタイマーを繰り返します。
4. 必要に応じて「一時停止」「停止」「1秒±シフト」「リセット」を使います。

## 主な機能

- 残り時間を `HH:MM:SS` 形式で大きく表示
- 終了10秒前から視覚表示と電子音でカウントダウン
- 周期終了音と即時の次周期開始
- 1秒単位のプラス／マイナスシフト
- 音量調整とミュート
- 時間設定、音量、ミュート状態を `localStorage` に保存
- ダークモード自動対応
- キーボード操作
  - Space: スタート／一時停止／再開
  - Escape: 停止
  - → または `+`: 1秒＋シフト
  - ← または `-`: 1秒－シフト

## 時刻計算について

残り時間は画面更新回数ではなく、周期の終了予定時刻と現在時刻の差から毎回計算します。次周期の境界も基準開始時刻、周期番号、累計シフトから算出するため、ブラウザの処理遅延が次周期へ累積しにくい設計です。

## ブラウザ制約

ブラウザがバックグラウンドになった場合や端末がスリープした場合、JavaScriptの実行や音声再生は制限されることがあります。復帰時には現在時刻から周期番号と残り時間を再計算しますが、バックグラウンド中やスリープ中に音を正確な時刻に鳴らすことは保証できません。
